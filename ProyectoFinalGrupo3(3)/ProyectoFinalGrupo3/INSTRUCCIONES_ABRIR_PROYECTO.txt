PROYECTO: ProyectoFinalGrupo3

Correcciones incluidas:
1. El proyecto mantiene el mismo nombre: ProyectoFinalGrupo3.
2. Se mantiene compatibilidad amplia: minSdk = 23, targetSdk = 35, compileSdk = 35.
3. Se corrigió el error de Android resource linking failed del ícono launcher:
   - Los adaptive icons fueron colocados en res/mipmap-anydpi-v26.
   - Los íconos normales .webp se mantienen en mipmap-mdpi/hdpi/xhdpi/xxhdpi/xxxhdpi.
4. Se mantiene la pantalla HU-003 Entradas y Salidas.

Cómo abrir:
1. Descomprime este ZIP.
2. Abre Android Studio.
3. File > Open.
4. Selecciona la carpeta ProyectoFinalGrupo3.
5. Espera el Sync.
6. Ejecuta con Run.

Importante:
No abras la carpeta app sola. Abre la carpeta principal ProyectoFinalGrupo3.
